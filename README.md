
# PatYdiomas V1

Um aplicativo web interativo para aprendizado de idiomas, inspirado no estilo gamificado do Duolingo. Desenvolvido com React e JavaScript.



## Funcionalidades

- **Interface gamificada**: Níveis (Fácil, Médio, Difícil) com 3 fases cada
- **Perguntas dinâmicas**: 5 perguntas por fase, selecionadas aleatoriamente de um banco de questões
- **Sistema de vidas**: 5 vidas iniciais, perde 1 a cada erro
- **Feedback sonoro**: Sons de acerto e erro para uma experiência mais imersiva
- **Progresso persistente**: Acompanhamento das fases completadas durante a sessão
- **Design responsivo**: Interface adaptável para desktop e mobile com Bootstrap
- **Sem necessidade de login**: Comece a jogar imediatamente inserindo seu nome



## Tech Stack

- **React** - Biblioteca para construção da interface
- **JavaScript** - Lógica do jogo
- **Bootstrap** - Estilização e componentes responsivos
